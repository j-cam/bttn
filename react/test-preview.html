React Usage

These CSS classes are automatically applied by the React <Button> component:

<Button variant="stroke" size="lg">Click me</Button>
// Outputs: <button class="bttn-stroke-lg">Click me</button>

<Button size="sm">Small</Button>
// Outputs: <button class="bttn-sm">Small</button>

<Button variant="ghost">Ghost</Button>
// Outputs: <button class="bttn-ghost">Ghost</button>
